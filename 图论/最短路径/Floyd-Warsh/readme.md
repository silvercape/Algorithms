# 计算最短路径

## 问题描述

计算任意两点间的最短路径

## 输入

第一行3个整数 n,m,q 表示 n 个点，m 条边，q 个查询 第2 ~ m+1行每行3个整数u，v, w，表示u，v之间存在一条长度为w的边 第 m+2~m+q+1行中，每行两个整数 s和t，查询s和t之间的最短路径长度。

```c++
4 2 2
0 1 2
1 3 4
0 3
0 2
```

## 输出

输出q行，每行对应一个查询， 若无法从 s 到达 t， 输出 -1。

## 数据说明

```c++
1 <= n <= 400
1 <= m <= n^2 /2
q <= 10^3
1 <=u,v,s,t<=n
1<=w<=10^9
```
